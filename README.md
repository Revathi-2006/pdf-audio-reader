# PDF Audio Reader

A simple Python desktop application that converts text-based PDF files into speech.

## Features

* Open PDF files using a graphical interface.
* Read PDF content aloud.
* Start and stop reading anytime.
* Simple and lightweight application.
* Built using Tkinter, PyPDF2, and pyttsx3.

## Limitations

* Supports only PDFs containing selectable text.
* Scanned/image-based PDFs are not supported in the current version.

## Technologies Used

* Python
* Tkinter
* PyPDF2
* pyttsx3
* Threading

## Installation (For Developers)

Clone the repository:

```bash
git clone <repository-url>
cd <repository-name>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python gui.py
```

## Using the Application

1. Download the application ZIP file from the Google Drive link provided below.
2. Extract the ZIP file.
3. Double-click `gui.exe`.
4. Click **Open PDF** and select a PDF file.
5. Click **Start Reading** to begin listening.
6. Click **Stop Reading** to stop the speech.
